# MEMO
簡易備忘錄，快速紀錄你一天的備忘事項。

## 主要功能
* 登入系統
* 即時儲存資料至雲端資料庫
* 添加備忘事項(自動按時間排序)


<img width="1440" alt="homepage" src="https://github.com/grace02468/MEMO_React/assets/85739113/728a1150-57ab-4a68-b7e6-0bb992630fce">

## 運用技術
* React
    * React Router
* Firebase
    * Cloud Firestore
    * Firebase Hosting
    * Firebase Authentication
